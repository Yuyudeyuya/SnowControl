# 🥵下雪 for Typecho

本插件实现对于Typecho前端的下雪效果

## 预览地址
[本人blog](https://blog.kokowo.cn/)
[feng的blog](https://bkfeng.vvvv.ee/)
## 功能

- 实现对于Typecho前端的下雪效果


## 使用方法
1. 在 Typecho的插件文件夹中添加本插件
2. 后台“插件管理”中找到本插件启用即可
3. 启用后自动注入到前端页面，无需手工注入
